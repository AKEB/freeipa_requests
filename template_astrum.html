Привет!

Подготовили подробный гайд для настройки доступа в новые сервисы Astrum Entertainment — он поможет тебе разобраться, как устроена авторизация для почты, мессенджера и внутренних систем.

Что важно знать в первую очередь:

  • Для авторизации в почте используется уникальный пароль, который предназначен только для почтового сервиса Astrum Entertainment.
  • Для авторизации во внутренних системах будет использоваться двухфакторная аутентификация, ниже можно найти инструкцию и одноразовые ссылки.
  • Для входа в мессенджер используется короткий одноразовый пароль, который генерируется в момент авторизации и отправляется вам на почту.

— Авторизация в корпоративном почтовом аккаунте Astrum Entertainment на базе Mail.ru

Адрес почтового ящика имеет следующий вид: user.name@{2} (обычно вида name.surname).

user.name и password доступны по ссылке - {3} (Внимание! Ссылка одноразовая!).
При первом открытии ссылки обязательно запомните user.name и password!

  • Для авторизации в браузере на сайте mail.ru наберите в окне авторизации полное наименование аккаунта вместе с окончанием @{2}, нажмите кнопку “Ввести пароль”;
  • Введите пароль от почты, который был вами получен ранее, и нажмите “Войти”;
  • Далее потребуется привязать свой номер телефона для двухфакторной аутентификации (с использованием одноразового цифрового пароля, получаемого по СМС).

Также почта может быть открыта в любом браузере и настроена с помощью службы техподдержки в MS Outlook и других почтовых приложениях, в том числе и мобильных. При настройке корпоративного почтового аккаунта в стороннем приложении, например, на мобильном устройстве, используйте пароль для внешнего приложения

— Настройки новой двухфакторной аутентификации для внутренних сервисов Astrum Entertainment

Ниже можно найти одноразовые ссылки c паролем и секретом для токена OTP, которые потребуются для аутентификации в https://jira.astrum.team и остальные сервисы.

  1. Скачайте приложение Google Authenticator в AppStore или Google Play.

  2. Войдите в приложение и нажмите на плюсик в правом нижнем углу. Далее выберете пункт “Сканировать QR-код” (Scan a QR code). Отсканируйте QR-код по этой ссылке - {0} (Внимание! Ссылка одноразовая!).
    Теперь в приложении у вас появилась строка с одноразовым паролем, который изменяется каждый 30 секунд (это аналог пароля на физическом токене, только в приложении). Называется этот пароль - OTP (One Time Password).

  3. Откройте страницу https://jira.astrum.team и авторизуйтесь в Jira. Используйте адрес вашего нового почтового ящика, выберите кнопку Login with astrum.team.
    Адрес почтового ящик имеет следующий вид: user.name@{2} (обычно вида name.surname).
    user.name и password доступны по ссылке - {1} (Внимание! Ссылка одноразовая!). При первом открытии ссылки обязательно запомните user.name и password!

  4. Далее откроется страница аутентификации astrum.team, которая будет использоваться для входа во все корпоративные сервисы astrum.team

  5. В поле Username or email введите ваш user.name, который вы получили по одноразовой ссылке выше. Он совпадает с левой частью вашего почтового адреса.

  6. В поле Password введите пароль, который состоит из двух частей:
    password (который вы получили по одноразовой ссылке выше) + ОТР (из приложения Google Authenticator).

  7. Нажмите Sign In и войдите в ваш корпоративный аккаунт.

— Авторизация в новом мессенджере VK Teams в инфраструктуре Astrum Entertainment

Обращаем внимание, что в мессенджере VK Teams нет возможности войти сразу в два аккаунта (corp.mail.ru и astrum.team). В течение переходного периода рекомендуем использовать для коммуникации с коллегами из Astrum Entertainment Desktop-версию, а для прежнего аккаунта (corp.mail.ru) WEB-версию. На мобильном устройстве необходимо будет перелогиниться в аккаунт Astrum Entertainment.

Ссылки для скачивания и установки приложения:

  • WEB-версия – https://myteam.mail.ru/webim/
  • iOS – https://itunes.apple.com/ru/app/id1459096573?mt=8
  • Android – https://play.google.com/store/apps/details?id=ru.mail.biz.avocado
  • Windows – https://vkteams-www.hb.bizmrg.com/win/x32/vkteamssetup.exe
  • Mac – https://vkteams-www.hb.bizmrg.com/mac/x64/vkteams.dmg
  • Linux – https://vkteams-www.hb.bizmrg.com/linux/x64/vkteams.tar.xz

Для входа в мессенджер сделайте следующее:

  • В WEB-версии нажмите на кнопку “Принять и продолжить” и введите Username как для почты, нажмите кнопку “Войти”. Далее введите одноразовый пароль, который в этот момент отправился вам на почту.

  • В приложениях для смартфонов и компьютеров сначала нужно будет разлогиниться из текущего аккаунта (Настройки -> Аккаунт -> Выйти), затем нажать на кнопку “Принять и продолжить”, ввести Username как для почты и нажать на кнопку “Войти”. Далее введите одноразовый пароль, который в этот момент отправился вам на почту.

Напомним, в случае возникновения проблем, вы всегда можете обратиться к нам:

  • поставить задачу в https://jira.astrum.team на проект "Техническая поддержка сотрудников (SUPPORT)";

  • создать задачу по прямой ссылке в Jira с выбранным типом задачи "Запрос в службу поддержки";

  • написать письмо с описанием проблемы на адрес tech.support@astrum.team.
